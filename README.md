# PC Tech Shop

Интернет-магазин компьютерной техники.

## Описание
Веб-приложение для управления магазином компьютерной техники, включая каталог товаров, корзину покупок и управление пользователями.

## Технологии
- Frontend: React.js
- Backend: Node.js
- База данных: MongoDB
